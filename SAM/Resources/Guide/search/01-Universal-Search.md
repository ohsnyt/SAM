# Universal Search

Search across people, notes, evidence, insights, and outcomes all at once. Universal search is the fastest way to find anything in SAM.

## Accessing Search

| Method | How |
|--------|-----|
| **Sidebar** | Click Search (⌘5) |
| **Command palette** | Press **⌘K** from anywhere in SAM |

## What's Searchable

| Type | What's Searched | Icon |
|------|----------------|------|
| **People** | Name, email, role | Person icon with role badge |
| **Contexts** | Context name | Building icon |
| **Notes** | Full text of note content and summaries | Orange note icon |
| **Evidence** | Titles, snippets from emails, meetings, messages, calls | Source-specific icon (calendar, mail, etc.) |
| **Insights** | Title and message text | Yellow lightbulb |
| **Outcomes** | Title and coaching rationale | Kind-colored dot |

## Filtering by Scope

Use the scope picker above results to narrow your search:

- **All** — Search everything (default)
- **People** — Only contacts
- **Contexts** — Only business contexts
- **Notes** — Only your captured notes
- **Evidence** — Only interaction evidence
- **Insights** — Only AI-generated insights
- **Outcomes** — Only coaching outcomes

## Results and Detail View

Search uses a two-column layout:

- **Left pane** — Search field, scope picker, and results grouped by type
- **Right pane** — Full detail view of the selected result

Click any result to see its complete details in the right pane. For people, this shows the full person detail view. For notes, you'll see the summary, content, linked people, and extracted action items.

## Search Behavior

- **Instant** — Results appear as you type with a 300ms debounce
- **Case-insensitive** — Searches match regardless of capitalization
- **Scope-aware** — Changing scope re-runs the search immediately
- **Full-text** — Searches body content, not just titles

## Tips

- **⌘K** is the fastest way to search from anywhere — you don't need to navigate to the Search section first
- Start broad (All scope), then narrow with the scope picker if you get too many results
- Search for a person's name to quickly pull up their profile from any context
- Evidence search is great for finding "that email from last week" — search by subject or sender name
