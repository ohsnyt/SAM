# Command Palette

The command palette is a Spotlight-style quick launcher that lets you navigate anywhere in SAM, find people, and run actions — all without touching the mouse.

## Opening the Command Palette

Press **⌘K** from anywhere in SAM. A floating search panel appears centered on screen.

## What You Can Do

### Navigate to Sections

Type a section name to jump directly:

| Command | Where It Goes |
|---------|--------------|
| "today" | Today view |
| "people" | Contact list |
| "business" | Business dashboard |
| "grow" | Grow dashboard |
| "events" | Event manager |
| "search" | Universal search |
| "settings" | Settings window |
| "graph" | Relationship graph |
| "guide" | SAM Guide |

### Find People

Start typing a person's name and SAM searches your contacts in real time. Select a person to navigate directly to their detail view.

### Run Actions

The palette includes quick actions:

| Action | What It Does |
|--------|-------------|
| "New note" | Opens the note capture area |
| "Import contacts" | Triggers a contact import |
| "Refresh briefing" | Regenerates your daily briefing |
| "Capture clipboard" | Opens clipboard capture (same as ⌃⇧V) |

## Keyboard Navigation

- **Up/Down arrows** — Move through results
- **Return** — Execute the selected item
- **Escape** — Close the palette

## How Search Works

The palette uses fuzzy matching — you don't need to type exact names or commands. Typing "biz" will match "Business dashboard," and typing part of someone's name finds them quickly.

Results are grouped by type: navigation commands appear first, followed by people matches.

## Tips

- **⌘K** is the fastest way to get anywhere in SAM — faster than clicking through the sidebar
- Use it to quickly pull up a person's profile when someone calls or messages you
- The palette remembers nothing between uses — it's a pure launcher, not a history
