# Text Size

SAM lets you adjust the text size throughout the app to match your display and preferences. This is especially useful on high-resolution displays where the default macOS text size may feel small.

## Changing Text Size

1. Open **Settings** (⌘,)
2. Go to the **General** tab
3. Find the **Text Size** section
4. Choose from four options:

| Size | Scale | Best For |
|------|-------|---------|
| **Small** | 88% | High-density workflows, large displays |
| **Standard** | 100% | Default macOS text size |
| **Large** | 115% | Easier reading, presentations |
| **Extra Large** | 130% | Accessibility, high-DPI displays at native resolution |

A preview sentence below the picker shows how text will appear at your chosen size.

## How It Works

SAM applies a custom text scale multiplier to all semantic fonts throughout the app — body text, headings, captions, and labels all scale proportionally. This ensures the visual hierarchy stays consistent at every size.

The text size change takes effect immediately across all views without restarting SAM.

## Tips

- If you've set a Retina display to its native resolution (maximum space), try Large or Extra Large for comfortable reading
- Text size is independent of macOS System Settings display scaling — it only affects SAM
