# Keyboard Shortcuts

SAM supports standard macOS keyboard shortcuts plus navigation shortcuts for fast access to every section.

## Navigation

| Shortcut | Action |
|----------|--------|
| ⌘1 | Go to Today |
| ⌘2 | Go to People |
| ⌘3 | Go to Business |
| ⌘4 | Go to Grow |
| ⌘5 | Go to Search |
| ⌘K | Command Palette (search from anywhere) |
| ⌘? | Open SAM Guide |

## Text Editing

These shortcuts work in every text input throughout SAM — notes, drafts, descriptions, and any multi-line editor.

| Shortcut | Action |
|----------|--------|
| ⌘S | Save current note or edit |
| ⌘N | New note |
| ⌘B | Bold |
| ⌘I | Italic |
| ⌘Z | Undo |
| ⇧⌘Z | Redo |
| ⌘F | Find / Search |

## Actions

| Shortcut | Action |
|----------|--------|
| ⌘⏎ | Execute primary action on focused item (save, send, confirm) |
| ⌃⇧V | Capture clipboard conversation as evidence |
| Esc | Cancel or dismiss the current sheet |
| ⌘, | Open Settings |

## Relationship Graph

| Shortcut | Action |
|----------|--------|
| Click + drag | Pan the canvas |
| Scroll wheel | Zoom in and out |
| Shift+Click | Add node to selection |
| Shift+Arrow | Navigate to connected nodes |
| Double-click | Open person's detail view |
| Escape | Clear selection |

## Tips

- **⌘K** is the fastest way to find anything — people, notes, evidence, outcomes — from anywhere in SAM
- **⌃⇧V** works globally (when Accessibility permission is granted), so you can capture conversations without switching to SAM first
- **⌘⏎** on an outcome card executes its primary action (write note, draft message, etc.)
