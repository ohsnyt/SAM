# Approving Summaries on iPhone

Once a recording's summary lands on iPhone, you sign off on it right there — no need to walk back to the Mac for routine approve-or-discard decisions.

## How to Use

1. After tapping Stop, the summary appears in the completed view (TL;DR, action items, decisions, follow-ups, etc.).
2. Review the summary. If something looks wrong, **reclassify** first (see "Reclassifying a Recording") before signing off.
3. Tap **Done** (green) when the summary looks right, or **Delete** (red, destructive) to discard the entire recording.
4. After Done, a brief "Done!" confirmation appears, then SAM clears the screen for the next meeting.

## What Done Does

- Marks the session as approved on the Mac.
- **Purges the audio file and verbatim transcript** from the Mac (compliance — see WFG retention policy).
- Keeps the AI-generated summary, action items, topics, and life events — those become the durable record.
- Hides **Change Recording Type** (no longer possible after sign-off).

## What Delete Does

- Permanently removes the recording, transcript, and summary from the Mac.
- Cannot be undone.
- A confirmation dialog asks before proceeding.

## Tips

- After Done, the **Record Another Meeting** button appears so you can start the next session without going back to a clean state.
- If the summary hasn't arrived yet, the **Done** button stays hidden until it does (or until the awaiting state times out). Delete is always available.
- If the Mac dropped offline mid-summary and you're stuck on "Generating summary…", **pull down to refresh** to clear the stale awaiting state.
