# Creating an Event

## Step by Step

1. Open the **Event Manager** (sidebar > Grow)
2. Make sure you're on the **Events** tab
3. Click the **+ New Event** button in the upper-right of the event list
4. Fill in the event details:

| Field | What to Enter | Example |
|-------|--------------|---------|
| **Title** | Name of your event | "Personal Finance Workshop for Employees" |
| **Description** | Brief summary (optional) | "90-minute interactive workshop covering budgeting, debt management, and retirement basics" |
| **Format** | In Person, Virtual, or Hybrid | In Person |
| **Start Date/Time** | When the event begins | March 25, 2026 at 6:30 PM |
| **End Date/Time** | When the event ends | March 25, 2026 at 8:00 PM |
| **Venue** | Location (for in-person/hybrid) | "Marriott Conference Room B" |
| **Join Link** | Video call URL (for virtual/hybrid) | https://zoom.us/j/123456 |
| **Target Attendance** | How many people you're aiming for | 25 |

5. Click **Create**

## Tips

- **Start with Draft status.** Every new event starts as a "Draft." This gives you time to add participants and prepare invitations before anything goes out.
- **You can edit everything later.** Don't worry about getting every detail perfect — you can update the event anytime.
- **Link a presentation** if you're using one of your standard decks (we'll cover this in the Presentations section).

## Event Lifecycle

Your event moves through these stages:

**Draft** > **Inviting** > **Confirmed** > **In Progress** > **Completed**

You control when to advance the status. SAM won't change it automatically — it's your call when invitations are done, when the event is confirmed, etc.

If plans change, you can also mark an event as **Cancelled**.
