# Linking Presentations to Events

## Why Link Them

When you link a presentation to an event, SAM can:

- Use the presentation content to write better, more specific invitations
- Suggest marketing copy based on the actual talking points
- Track how many times you've delivered each presentation
- Show delivery history — when you last gave it and when it's next scheduled
- Help you plan your event calendar around your presentation rotation

## How to Link

When creating or editing an event, you can select a presentation from your library to associate with it. This creates a two-way connection:

- **From the event:** You can see which presentation is being used
- **From the presentation:** You can see all events where it's been or will be delivered

## Delivery Tracking

Once linked, the presentation detail view automatically shows:

- **Delivery Count** — How many completed events used this presentation
- **Last Delivered** — The most recent completed event date
- **Next Scheduled** — The soonest upcoming event date

This helps you plan your rotation. If you have 3-6 presentations you cycle through, you can quickly see which ones haven't been delivered recently and are due for another round.

## Using Presentation Content for Communications

The content SAM extracts from your presentations feeds directly into invitation and promotion drafts. When SAM writes an invitation for an event that has a linked presentation, it references the actual topics covered — not generic filler.

For example, instead of:
> "Join us for an informative financial workshop"

SAM writes:
> "Join us for a hands-on workshop covering the 50/30/20 budgeting rule, debt payoff strategies, and how to maximize your employer's 401(k) match"

The specificity comes from SAM having read your actual slides.
