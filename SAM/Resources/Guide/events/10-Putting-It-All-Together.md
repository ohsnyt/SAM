# Putting It All Together — A Typical Workflow

## Sarah's Workshop Workflow

Here's how everything fits together for a typical recurring workshop:

### One Time: Set Up Your Presentations

1. Drag your 3-6 workshop PDFs into the **Presentations** tab
2. Wait for SAM to digest each one (watch the minions in the sidebar)
3. Review the summaries and talking points SAM extracted
4. Add descriptions and target audience details if needed

### 2-3 Weeks Before: Create the Event

1. Go to **Events** tab, click **+ New Event**
2. Fill in the details: title, date, time, venue
3. Link the appropriate presentation from your library
4. Add your initial participant list using **Add People**
5. Set VIP/Key priority for speakers, co-hosts, or important guests

### 2 Weeks Before: Send Invitations

1. Click **Draft Invitations** to generate personalized messages for everyone
2. Review SAM's drafts — the content will reference your actual presentation topics
3. Edit anything that needs a personal touch
4. Copy and send via email, text, or your preferred channel
5. Click **Promote** to draft social media posts for LinkedIn, Facebook, etc.

### 1-2 Weeks Before: Monitor RSVPs

1. SAM automatically tracks responses from your messages and emails
2. Check the participant list — statuses update as responses come in
3. Review any auto-added participants (people who heard about it and texted you)
4. Confirm any low-confidence RSVP detections SAM flagged
5. Send follow-up reminders to anyone who hasn't responded

### Day of the Event

1. Review your participant list — you know exactly who to expect
2. Update the event status to **In Progress**
3. After the event, mark attendance for each participant
4. Update status to **Completed**

### After the Event

1. The presentation's delivery count increments automatically
2. SAM can help you draft follow-up messages to attendees
3. Start planning the next event — your presentation rotation is tracked

## The Feedback Loop

The more you use events, the smarter SAM gets:

- **Invitation style** — SAM learns your preferred wording and sign-off
- **RSVP patterns** — SAM gets better at matching responses to events
- **Participant suggestions** — SAM learns which types of contacts attend which types of events
- **Timing** — SAM notices which days and times get the best attendance

## Quick Reference

| Task | Where | Button/Action |
|------|-------|---------------|
| Create event | Events tab | + New Event |
| Add participants | Event detail | Add People |
| Send invitations | Event detail | Draft Invitations |
| Send one invitation | Participant row | Draft Invitation (Quick Action) |
| Promote on social | Event detail | Promote |
| Add presentation | Presentations tab | Drag PDF or + New |
| Check RSVP status | Event detail | Participant list (auto-updates) |
| Undo deletion | Toast notification | Click Undo |
| Set your name/closing | Settings > General | Your Identity section |
